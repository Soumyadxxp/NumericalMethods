# Numerical Methods in C

A comprehensive collection of Numerical Methods algorithms implemented in **C Programming**. This repository serves as a practical reference for engineering, computer science, and mathematics students studying Numerical Analysis.

Each program is written with simplicity, readability, and educational value in mind, making it suitable for academic coursework, laboratory assignments, and self-learning.

---

## Overview

Numerical Methods provide computational techniques for solving mathematical problems that cannot be solved analytically or efficiently by traditional methods.

This repository contains implementations of some of the most commonly used numerical algorithms including:

- Root Finding Methods
- Interpolation Methods
- Numerical Integration
- Differential Equation Solvers
- Iterative Linear Equation Solvers

All programs are written in standard C and can be compiled using GCC, Dev-C++, Code::Blocks, Turbo C, or any modern C compiler.

## Compilation

Using GCC:

```bash
gcc filename.c -o output
```

Example:

```bash
gcc "Bisection Method in.c" -o bisection
```

Run the executable:

### Windows

```bash
bisection.exe
```

### Linux / macOS

```bash
./bisection
```
